# 21. Merge Two Sorted Lists

Merge two sorted linked lists and return it as a new list. The new list should be made by splicing together the nodes of the first two lists.

**Example:**

```
Input: 1->2->4, 1->3->4
Output: 1->1->2->3->4->4
```



## Code:

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def mergeTwoLists(self, l1, l2):
        """
        :type l1: ListNode
        :type l2: ListNode
        :rtype: ListNode
        """
        dummy = ListNode(0)
        cur = dummy
        p1 = l1
        p2 = l2
        while p1 and p2:
            if p1.val > p2.val:
                cur.next = p2
                cur = cur.next
                p2 = p2.next
            else:
                cur.next = p1
                cur = cur.next
                p1 = p1.next
        while p1:
            cur.next = p1
            cur = cur.next
            p1 = p1.next
        while p2:
            cur.next = p2
            cur = cur.next
            p2 = p2.next
        return dummy.next
```
